# Trainify

## Название и описание

**Trainify** — нативное Android-приложение для планирования тренировок, отслеживания прогресса и построения тренировочного процесса с помощью ИИ.

Проект решает типичную проблему пользователей фитнес-приложений: разрозненность инструментов — отдельно дневник тренировок, отдельно таблица с весами и повторениями, отдельно поиск техники выполнения упражнений на YouTube, отдельно трекер прогресса тела. Trainify объединяет всё это в одном приложении и добавляет слой ИИ-автоматизации: генерацию упражнений, шаблонов и планов тренировок, распознавание тренажёров по фото и анализ техники выполнения упражнения по видео.

Приложение рассчитано на людей, занимающихся в тренажёрном зале или дома, которые хотят вести подробный журнал тренировок (подходы, повторения, вес, дроп-сеты, кластерные подходы и т.д.), отслеживать динамику веса тела и получать помощь ИИ при составлении программы тренировок.

## Основные функции

### Тренировки и планирование
- Создание тренировки вручную с выбором упражнений из каталога.
- Поддержка разных типов подходов: обычные, дроп-сеты, кластерные подходы.
- Таймер тренировки, работающий в foreground-сервисе, заметки к тренировке, календарь тренировок.
- Шаблоны тренировок для повторного использования программ.
- Планирование тренировок по дням и составление тренировочного плана.
- История тренировок с группировкой по неделям/месяцам/годам и детальным просмотром.
- Аналитика по упражнениям: динамика объёма, времени, рекордов.

### ИИ-функции (Google Gemini)
Все ИИ-запросы идут через прокси на Cloudflare Worker:
- **Генерация упражнения** по описанию, группе мышц, оборудованию и уровню сложности.
- **Генерация шаблона тренировки** целиком.
- **Генерация плана тренировки на день** и **генерация многодневного тренировочного плана**.
- **Подсказки при выборе упражнений**.
- **Распознавание тренажёра/оборудования по фотографии** камеры.
- **Анализ техники выполнения упражнения по видео**: локальная предобработка видео (Media3) перед отправкой в Gemini.
- **Поиск обучающих видео по технике на YouTube** через YouTube Data API (с приоритетом определённых каналов, ранжированием по релевантности).
- Все длительные ИИ-генерации могут выполняться в фоне через foreground-сервис с уведомлением о прогрессе.

### Прогресс и геймификация
- Трекер прогресса тела: вес, фото прогресса.
- Система наград/достижений с уровнями **Wood → Bronze → Silver → Gold → Platinum** по 14 категориям: общее число тренировок, суммарный тоннаж, суммарные повторения, шаги в день, серия тренировок подряд (streak), недельная стабильность, персональные рекорды по конкретным упражнениям, использование ИИ-генерации, подключение Health Connect, использование распознавания оборудования и др.

### Интеграции с устройствами
- **Google Health Connect**: чтение шагов, пульса и потраченных калорий.
- **Bluetooth**-подключение к внешним устройствам.

### Аккаунт и синхронизация
- Аутентификация через Firebase Authentication.
- Локальное хранение данных в Room с синхронизацией в Firebase Firestore.

### Уведомления и настройки
- Локальные уведомления: напоминание о тренировке и напоминание о взвешивании (WorkManager).
- Настройки темы оформления, параметров тренировок, уведомлений и системных разрешений.

## Используемые технологии

**Мобильное приложение (Android, Kotlin):**
- **Язык:** Kotlin
- **UI:** Jetpack Compose (Material 3)
- **Архитектура:** MVVM (`ViewModel` + `Repository`)
- **Локальная БД:** Room
- **Хранилище настроек:** DataStore
- **Сеть:** Retrofit2, а также нативный `HttpURLConnection`
- **Изображения:** Coil Compose
- **Фоновые задачи:** WorkManager, foreground-сервисы
- **Видео:** AndroidX Media3 — предобработка видео перед анализом техники
- **Здоровье:** AndroidX Health Connect `1.1.0`
- **Firebase:** Auth, Firestore
- **Аутентификация:** Firebase Auth
- **Cloudflare Worker:** Google Gemini API(`gemini-2.5-flash`,`gemini-3.1-flash-lite`, и др.), YouTube Data API V3

**Прочее:**
- Локализация: `en`, `ru`, `be`.
